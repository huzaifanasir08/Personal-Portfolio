# **Personal Portfolio**

This is a responsive and modern portfolio website showcasing my skills, projects, and ways to contact me. The site is built using **Bootstrap** for styling and includes a backend for fetching dynamic data through APIs.

## **Features**

- **About Section** ‚Äì Highlights my experience, background, and key skills.
- **Projects Section** ‚Äì Displays the projects I've worked on, with real-time data fetching for updates.
- **Contact Page** ‚Äì Provides a contact form that allows visitors to reach out directly.
- **Backend Integration** ‚Äì Uses Axios to fetch project data dynamically from an API.
- **Custom Styling** ‚Äì Features a modified color scheme for a unique and modern appearance.
- **Mobile Responsive** ‚Äì Fully optimized for all screen sizes using Bootstrap's responsive grid system.
- **Smooth Animations** ‚Äì Includes subtle animations and hover effects to enhance user experience.
- **SEO Friendly** ‚Äì Structured with best practices to improve search engine visibility.

## **Tech Stack**

### **Frontend**
- **HTML5** ‚Äì Markup language for structuring the web pages.
- **CSS3** ‚Äì Styling the UI with custom themes and Bootstrap framework.
- **JavaScript (ES6+)** ‚Äì Adds interactivity and dynamic features.
- **Bootstrap 5** ‚Äì Provides a responsive and modern design framework.
- **Axios** ‚Äì Handles API requests for fetching data from the backend.

- **Django** ‚Äì Manages backend logic and API endpoints.

## **Setup & Installation**

1. Clone the repository:
   ```sh
   git clone https://github.com/YOUR_GITHUB_USERNAME/Personal-Portfolio.git
   ```  
2. Navigate to the project directory:
   ```sh
   cd Personal-Portfolio
   ```  
3. Install dependencies (if applicable):
   ```sh
   npm install
   ```  
4. Run the project in development mode:
   ```sh
   npm start
   ```  
5. Build for production:
   ```sh
   npm run build
   ```
